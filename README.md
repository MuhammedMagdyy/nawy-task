## 📦 Installing

1. **Clone the repository** 🔗:

   ```bash
   git clone https://github.com/MuhammedMagdyy/nawy-task.git
   ```

2. **Navigate to the project directory** 📁:

   ```bash
   cd nawy-task
   ```

3. **Create a `.env` file** 📝:

   ```bash
   cp .env.example .env
   ```

4. **Start the server** 🚀:

   ```bash
   docker-compose up --build
   ```
